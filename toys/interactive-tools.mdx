# Interactive & Creative Tools

<Info>

</Info>

## Interactive Tool Features

<AccordionGroup>
  <Accordion title="Pinch & Rotate Gestures">
    Many interactive tools support **pinch and rotate gestures** for real-time control:
    
    - **Pinch**: Scale, zoom, or adjust density
    - **Rotate**: Twist, spin, or change flow direction
    
    Supported toys:
    - Aurora Painter
    - Bioluminescent Tidepools
    - Spiral Burst
    - Star Field
    
    <Tip>
      On desktop, use mouse drag + modifier keys (Shift/Alt) to simulate pinch/rotate.
    </Tip>
  </Accordion>
  
  <Accordion title="Arrow-Key Mode Switching">
    Interactive tools often include **arrow-key mood switching** to cycle through preset visual modes:
    
    - **Left/Right arrows**: Cycle forward/backward through moods
    - **Up/Down arrows**: Adjust intensity or speed
    
    Supported toys:
    - Aurora Painter
    - Bioluminescent Tidepools
    - Spiral Burst
    - Star Field
  </Accordion>
  
  <Accordion title="Device Motion & Tilt">
    Some toys respond to **device motion sensors** (accelerometer/gyroscope):
    
    - **Tactile Sand Table**: Tilt to shift gravity direction and sand flow
    - **Multi-Capability Visualizer**: Device motion drives camera and shape movement
    
    <Info>
      Device motion requires user permission on iOS and recent Android versions. The toy will prompt when needed.
    </Info>
  </Accordion>
  
  <Accordion title="Touch Drift & Quality">
    Mobile-optimized toys include **touch drift** controls to add subtle motion without constant touch input, plus **quality presets** to balance visual fidelity with battery life:
    
    - **Pocket Pulse**: Touch drift adds gentle pulse motion
    - **Mobile Ripples**: Touch drift creates continuous ripple waves
    
    Quality presets include:
    - **Low**: Minimal particles/effects for maximum battery life
    - **Medium**: Balanced visuals for mid-tier devices
    - **High**: Full fidelity for flagship devices
  </Accordion>
</AccordionGroup>

## Getting Started with Interactive Tools

<Steps>
  <Step title="Enable Touch/Motion Permissions">
    Some tools require **device motion** permission. The toy will prompt when needed.
    
    <Tip>
      On iOS, you may need to enable motion in Settings → Safari → Motion & Orientation Access.
    </Tip>
  </Step>
  
  <Step title="Try Gestures First">
    Start with **pinch/rotate gestures** or **arrow-key mode switching** to see immediate changes before diving into control panels.
  </Step>
  
  <Step title="Use Starter Presets">
    Featured tools include **starter presets** that bundle optimal settings for first-time users.
  </Step>
  
  <Step title="Adjust Quality for Mobile">
    Mobile toys include **quality presets**—use Low or Medium if you notice dropped frames or battery drain.
  </Step>
</Steps>

## Creative Workflows

<CardGroup cols={2}>
  <Card title="Painting Workflow" icon="palette">
    **Aurora Painter** and **Bioluminescent Tidepools** support continuous sketching:
    
    1. Start demo audio or microphone
    2. Adjust glow/trail strength
    3. Use pinch/rotate to shape flow
    4. Switch moods with arrow keys
  </Card>
  
  <Card title="Sculpting Workflow" icon="hammer">
    **Pottery Wheel Sculptor** offers tool-based sculpting:
    
    1. Start with Smoothing tool
    2. Make broad strokes to define shape
    3. Switch to Carving tool for details
    4. Use Pinching tool for fine adjustments
  </Card>
  
  <Card title="Motion-Based Workflow" icon="compass">
    **Tactile Sand Table** and **Multi-Capability Visualizer** respond to device tilt:
    
    1. Enable device motion permission
    2. Start audio (microphone or demo)
    3. Tilt device to shift gravity/camera
    4. Adjust damping/sensitivity as needed
  </Card>
  
  <Card title="Touch-First Workflow" icon="hand">
    **Pocket Pulse** and **Mobile Ripples** optimize for touch:
    
    1. Start with Medium quality preset
    2. Tap/drag to create ripples/pulses
    3. Enable Touch drift for continuous motion
    4. Adjust quality if performance drops
  </Card>
